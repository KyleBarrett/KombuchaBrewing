### Running the App
* The app is deployed on shinyapps.io:  https://kylebarrett.shinyapps.io/Kombucha/
* If you have R installed, you can run the app locally: 
```
library(shiny)
runUrl("https://github.com/KyleBarrett/KombuchaBrewing/archive/v1.0.zip")
```
